# Activity 2: User Stories - GDM Chatbot
## CS 584 Case Study

**Instructions:** Write 3 user stories for each persona you created.

**Format:** As [persona name], I want [goal], So that [benefit].

---

## EXAMPLES

**Example 1:**
> As Urban Priya,
> I want to log my glucose with one tap and a photo of my meal,
> So that I can quickly track during my work lunch break.

**Example 2:**
> As Rural Lakshmi,
> I want to record my glucose using voice in Malayalam,
> So that I can track without typing, which is difficult for me.
